Privacy Policy

Effective date: July 14, 2026

T-Account Builder ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use the T-Account Builder Microsoft Excel add-in and the taccountbuilder.com website.

1. Information We Process

When you use T-Account Builder, your spreadsheet data (column headers and journal entry rows) is sent to our API for processing. This data is used solely to generate your T-account visualizations and is never stored, logged, or retained after your request is complete. Our API is stateless — no financial data is written to any database or file.

2. Information We Do Not Collect

  • We do not collect your name, email address, or any personal information through the add-in
  • We do not store your financial or general ledger data
  • We do not use cookies or tracking on this website beyond standard web server logs
  • We do not sell, rent, or share any data with third parties

3. Third-Party Services

T-Account Builder uses the following third-party infrastructure to deliver the service:

  • Render.com — hosts our API server. Data sent for processing passes through Render's infrastructure but is not stored. See Render's Privacy Policy.
  • GitHub Pages — hosts the add-in files served to your Excel application. No personal data is transmitted to GitHub Pages. See GitHub's Privacy Statement.
  • Microsoft AppSource — handles add-in installation and, if applicable, subscription billing. See Microsoft's Privacy Statement.

4. Data Security

All data transmitted between the Excel add-in and our API is encrypted in transit using HTTPS. We do not retain financial data after processing, which minimizes security risk.

5. Children's Privacy

T-Account Builder is intended for professional use by accountants and financial professionals. We do not knowingly collect information from children under 13.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated effective date. Continued use of T-Account Builder after changes constitutes acceptance of the updated policy.
